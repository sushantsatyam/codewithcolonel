# Day 03 — Tool routing (structured outputs)

This notebook practices tool routing by asking the model to select a tool and respond with JSON.

## What this project shows

- Simple tool definitions
- Prompting for structured JSON output
- Parsing and executing a tool call locally

   ```bash

   ```
2. Set `OPENAI_API_KEY` in your environment.
3. Open the notebook and run the cells.
